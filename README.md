## 🎯 About The Project

This is a responsive recipe page built as a frontend practice project. It demonstrates the use of semantic HTML, modern CSS styling, custom fonts, and clean layout design.

## ✨ Features

- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🔤 Custom Google Fonts integration
- 📋 Well-structured recipe layout
- 🥘 Ingredients list with custom bullets
- 📝 Step-by-step numbered instructions
- 📊 Nutrition information table
- 🎨 Aesthetic color palette

## 🛠️ Built With

- **HTML5** - Semantic markup
- **CSS3** - Styling and layout
- **Google Fonts** - Young Serif & Outfit
